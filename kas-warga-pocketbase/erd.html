📋 Contoh Flow Transaksi

💰 Iuran Warga → Kas

1
Buat transactions: type=IURAN, from_wallet=warga, to_wallet=kas
2
Buat ledgers DEBIT pada wallet warga
3
Buat ledgers CREDIT pada wallet kas
4
Update balance kedua wallet, set status=SUCCESS

🏗️ Pengeluaran Kas

1
Buat transactions: type=PENGELUARAN, from_wallet=kas, to_wallet=null
2
Buat ledgers DEBIT pada wallet kas
3
Update balance kas, set status=SUCCESS

↩️ Reversal / Koreksi

1
Set transactions lama status=REVERSED
2
Buat transactions baru: type=REVERSAL, parent_trx=transaksi_lama
3
Buat 2 ledger entry kebalikan dari transaksi asli
4
Rollback balance wallet terkait

📊 Audit Balance

1
Ambil semua ledger dengan wallet = X
2
Jumlahkan CREDIT - DEBIT = balance teoritis
3
Bandingkan dengan wallets.balance
4
Jika beda → ada anomali, cek ledger entry terakhir
